# MermaidLens Theme Template

Use this folder as the starting point for a new MermaidLens theme.

## How to create a theme

1. Copy this folder to `packages/themes/<your-theme-id>`.
2. Update `theme.json` with your theme metadata and Mermaid theme variables.
3. Keep the `id` unique and in lowercase.
4. Share your theme with the community by opening a pull request.
